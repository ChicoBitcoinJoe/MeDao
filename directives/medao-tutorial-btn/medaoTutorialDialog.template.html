Button Descriptions

send

Opens a panel to send ether or time invested in this MeDao.

pay

Opens a panel to make a payment if any time has been used. Time is burned to show proof of work.

receive

Opens a panel with a qr code to be scanned or a copy button of the current account address.

clones

Opens a panel that shows any clones that have been created for this MeDao. Also where the admin can create new clones or change a clones url.

exchange

*soon*

Opens a panel to allow exchanging time and ether in bundles of 1 hour

Admin Only

start week

Can only used per week!

Set your weekly hours to how much time you will devote to your medao each week. If you become overscheduled either set your weekly hours lower, work overtime, or both until your schedule is balanced again.

set vault

The vault address is originally set to the Founder's address

Open a panel to set a new vault address. The vault is where any ether or erc20 tokens will ultimately be transferred when sent to the MeDao. It is critical this address be a safe address that can send and receive ether and erc20 tokens. I recommend this multisig wallet by Consensys.

set webpage url

Opens a panel to set the webpage that details the terms of the MeDao Contract. Make sure you are very clear on your site regarding what potential shareholders are agreeing to when they invest in your time. Confusion will only hurt your reputation and self worth!

set controller

The controller is initially set to the Founder's address

Opens a panel to set a new controller for your medao. I recommend this multisig wallet by Consensys.
