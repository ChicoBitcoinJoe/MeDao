Button Descriptions

send

Opens a panel to send ether or time invested in this MeDao.

pay

Opens a panel to make a payment of spent time to be burned as proof of work.

receive

Opens a panel with a qr code to be scanned or a copy button of the current account address.

place bid

Opens a panel to place a bid at auction. Hint: click on an existing teir to fill that value into the input.

start auction

Clicking this will start auction process. Congratulations to the highest bidder!

remove bid

Opens a panel that shows any bids the current account has placed. Click the red 'x' to reclaim the ether from the bid.

Admin Only

set weekly hours

Can only be set once per week!

Set your weekly hours to how much time you will devote to your medao each week. If you become overscheduled either set your weekly hours lower, work overtime, or both until your schedule is balanced again.

set vault address

The vault address is originally set to the Founder's address

Open a panel to set a new vault address. The vault is where any ether or erc20 tokens will ultimately be transferred when sent to the MeDao. It is critical this address be a safe address that can send and receive ether and erc20 tokens. I recommend this multisig wallet by Consensys.

set burn minimum

The burn minimum is originally set to 1 second.

Opens a panel to set the burn minimum for making a payment. Set this to a higher amount if the comment section is spammed by low value burn transactions.

transfer ownership

The owner is originally set to the Founder's address

Opens a panel to set a new owner contract for your medao. It is recommended to use a multisig address (such as the one in 'set vault address'

disable/enable transfers

Toggles whether transfers are enabled or disabled. In the event of an update, clone, or hack/bug, it is smart to disabled transfers until the matter is resolves completely.

set webpage url

Opens a panel to set the webpage that details the terms of the MeDao Contract. Make sure you are very clear on your site regarding what potential shareholders are agreeing to when they invest in your time. Confusion will only hurt your reputation and self worth!
